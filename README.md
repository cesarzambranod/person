## Description
Application created to be able to cook recipes with the ingredients you have in your refrigerator

Knowing how many calories you are consuming

You can add units, quantity, ingredient and food recipes.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
